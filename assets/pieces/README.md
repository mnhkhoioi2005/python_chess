# Hình ảnh quân cờ

Đặt file hình ảnh quân cờ vào thư mục này với tên:

## Quân trắng:
- wK.png - Vua trắng
- wQ.png - Hậu trắng  
- wR.png - Xe trắng
- wB.png - Tượng trắng
- wN.png - Mã trắng
- wP.png - Tốt trắng

## Quân đen:
- bK.png - Vua đen
- bQ.png - Hậu đen
- bR.png - Xe đen
- bB.png - Tượng đen
- bN.png - Mã đen
- bP.png - Tốt đen

## Kích thước khuyến nghị:
- 64x64 pixels hoặc 128x128 pixels
- Format: PNG với nền trong suốt
- Chất lượng cao để hiển thị đẹp

Nếu không có file hình, game sẽ dùng ký tự Unicode mặc định.
